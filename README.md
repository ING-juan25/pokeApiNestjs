# Ejecutar en desarrolo 

1. Clonar el repositorio
2. Ejecutar
```
yarn install
```
3. Tener Nest CLI istando
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos
```	
docker-compose up -d
```
